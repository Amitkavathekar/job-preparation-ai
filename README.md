
# 🚀 Job Preparation AI (Full-Stack AI Platform)

An AI-powered full-stack web application designed to help users analyze resumes, identify skill gaps, and prepare for technical interviews effectively.

---

## 🌐 Live Demo

🔗 Frontend: https://job-preparation-ai.vercel.app
🔗 Backend API: https://job-preparation-ai-backend.onrender.com

---

## ✨ Features

* 📄 **AI Resume Analysis** – Upload resume and get intelligent insights
* 🧠 **Skill Gap Detection** – Identify missing skills based on job roles
* 🎯 **Interview Preparation** – AI-generated interview questions
* 🔐 **Authentication System** – Secure login/signup using JWT
* 📑 **Dynamic PDF Generation** – Generate resumes dynamically
* ⚡ **Responsive UI** – Fully optimized for mobile, tablet, and desktop

---

## 🛠️ Tech Stack
- Frontend: Next.js, TypeScript, Tailwind CSS, Shadcn UI
- Backend: Node.js, Express.js, MongoDB
- AI: Gemini API
